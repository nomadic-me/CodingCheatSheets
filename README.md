# Coding Cheat Sheets
## Git Commands
**To Update an existing / cloned Repo**
```bash
git pull origin main
```

**To add all files** 
```bash
git add .
```
**To commit files** 
```bash
git commit -a -s -m 'Commit Message here'
```

**To discard the local changes** 
```bash
git reset --hard
```
